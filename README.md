# 📝 Task List App

A simple and responsive task management web app built with **HTML**, **Bootstrap**, and **JavaScript**.  
This project allows users to manage a list of tasks using a hardcoded array — no backend required.

---

## 📁 Folder Structure

project/
├── index.html # Main HTML file
├── style.css # Custom styles
└── script.js # App logic

---

## 🚀 Features

- ✅ View a list of tasks  
- ➕ Add new tasks  
- ☑️ Mark tasks as completed or undo  
- ❌ Delete tasks  
- 🎨 Styled with Bootstrap + minimal custom CSS  

---

## 🔧 Tech Stack

- HTML5  
- Bootstrap 5  
- Vanilla JavaScript  
- Custom CSS

---

## 📦 How to Run Locally

1. **Clone** or **download** the repository.
2. Open `index.html` in your web browser.
3. No server or installation required.
